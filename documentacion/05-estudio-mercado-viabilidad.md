## 🎯 Resumen Ejecutivo

### La Oportunidad

Ataena CRM es un software de gestión para estudios de tatuajes con dos diferenciadores clave:
1. **100% Offline** - Los datos del cliente están en SU ordenador, no en la nube
2. **Pago único** - Sin suscripciones mensuales, licencia perpetua

### Origen del Proyecto

- Software desarrollado inicialmente para un estudio amigo
- Si se puede vender a otros, es un bonus

### Mercado Objetivo

- **Inicial:** Zona Madrid/Guadalajara
- **Futuro (si funciona):** España completa
- **Target:** Tatuadores individuales y estudios pequeños (1-2 artistas)

### Proyección REALISTA (Año 1 - Solo Madrid/Guadalajara)

| Escenario | Ventas/año | Ingresos | Beneficio |
|-----------|------------|----------|-----------|
| 🔴 Ultra-pesimista | 5 | 395€ | ~320€ |
| 🟡 Conservador | 10-15 | 790-1.185€ | ~700-1.050€ |
| 🟢 Si todo va bien | 25-30 | 1.975-2.370€ | ~1.750-2.100€ |

### Conclusión

| Como... | ¿Viable? |
|---------|----------|
| Negocio principal | ❌ NO |
| Ingreso extra significativo (500€+/mes) | ⚠️ Difícil |
| Side project con algo de retorno | ✅ SÍ |
| Software para amigo + si vendo algo, mejor | ✅ SÍ |

## 🎯 Posicionamiento de Ataena

### Propuesta de Valor

> **"El software de gestión para tatuadores que no te cobra cada mes"**
>
> Gestiona clientes, citas y consentimientos digitales.  
> Sin suscripciones. Sin nubes. Tus datos son tuyos.

### Target Detallado

| Característica | Descripción |
|----------------|-------------|
| **Quién** | Tatuador autónomo o estudio pequeño (1-2 personas) |
| **Dónde** | España (inicial), Latinoamérica (futuro) |
| **Problema** | Gestionar clientes, citas y consentimientos legales |
| **Frustración** | Suscripciones mensuales, datos en la nube de otros |
| **Deseo** | Software simple, pago único, control de sus datos |

### Mensajes Clave

| Para quién | Mensaje |
|------------|---------|
| Harto de suscripciones | "Paga una vez, tuyo para siempre" |
| Preocupado por privacidad | "Tus datos en TU ordenador, no en nuestra nube" |
| Internet poco fiable | "Funciona sin conexión, siempre" |
| Presupuesto ajustado | "79€ una vez < 150€/año para siempre" |

### Lo que NO eres

- ❌ No eres para estudios grandes (5+ artistas)
- ❌ No tienes app móvil
- ❌ No tienes recordatorios WhatsApp automáticos
- ❌ No eres solución "en la nube desde cualquier sitio"

## ⚠️ Riesgos y Mitigación

### Riesgos Identificados

| Riesgo | Probabilidad | Impacto | Mitigación |
|--------|--------------|---------|------------|
| **Pocos clientes interesados** | Media | Alto | Validar con el estudio de prueba, encuestas |
| **Competencia baja precios** | Baja | Medio | Ya eres el más barato en perpetuo |
| **Piratería** | Media | Bajo | Sistema de licencias básico |
| **Bugs/soporte excesivo** | Media | Medio | Buen testing, FAQ, documentación |
| **No hay diferenciación** | Baja | Alto | Offline + pago único es único |
| **Falta de tiempo para mantener** | Alta | Alto | Automatizar todo, cobrar por soporte |

